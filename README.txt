== Description ==

Interactive graphs and chart builder with 9 supported types

== Frequently Asked Questions ==

= How do I use it? =

A chart button is added in the TinyMCE toolbar. Clicking on it will open the popup window where you can design the graph. 

= How do I show the graph? =

A short-code is added to the page or post. Alternatively, you can copy that too. 

= Is it responsive? =

Yes, the graphs are responsive and will take the width of the container.
